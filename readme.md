# SDL2 Examples

## Development setup:

### Linux/Debian

    sudo apt-get install libsdl2-dev

Additional support libs:

    libsdl2-image-dev
    libsdl2-ttf-dev
    libsdl2-mixer-dev

## Building:

In each folder run `make` to build the executable.

## VSCode:

Make a copy of `.vscode/c_cpp_properties.example.json` and set the include paths for your system.
